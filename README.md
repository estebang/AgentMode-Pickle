# Pickleball Court Reservation App

This is a React single page application for tracking pickleball court reservations in a facility with 10 courts. The app supports daily and weekly views of reservations.

## Getting Started

1. Install dependencies:
   ```powershell
   npm install
   ```
2. Start the development server:
   ```powershell
   npm run dev
   ```
3. Open your browser to the local server URL shown in the terminal.

## Features
- View reservations for 10 courts
- Daily and weekly calendar views
- Built with React and Vite
